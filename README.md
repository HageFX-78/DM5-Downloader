# DM5 Chapter Downloader

[Click here](https://github.com/HageFX-78/DM5-Downloader/raw/refs/heads/main/DM5Downloader.user.js) to quick install (If you already have a userscript manager).

## About

This is a simple userscript that adds a download button for individual manga/manhua chapter as a zip. Batch download is also supported as of v1.1, requires popup setting set to allowed in your browser to use, turn off after use.

## Update

-   v1.1 : Added batch download through workaround (requires browser popup enabled while using)
